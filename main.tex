\begin{abstract}
\thispagestyle{plain}
\noindent An estimate of the sensitivity of the fully hadronic vector boson scattering (VBS) channel to anomalous quartic gauge couplings (aQGCs), using the ATLAS detector with $140\,\mathrm{fb}^{-1}$ of proton-proton collision data at $\sqrt{s}=13\,\mathrm{TeV}$ using machine learning methods, is presented. This final state carries the largest combined branching fraction of all VBS topologies but also the most overwhelming QCD multijet background, which a two-stage classifier chain is designed to overcome. A six-jet VBF-RNN tagger combines the forward tagging jets with the two large-$R$ signal jets to identify the VBS topology, raising the inclusive AUC from $0.760$ to $0.801$ over a four-jet baseline. A subsequent multilayer perceptron separates the aQGC signal from the composite background using twelve event-level features. Three discriminant architectures are implemented and compared: specialised binary classifiers trained per operator family, a global binary classifier spanning all $19$ probed operators of the Éboli dimension-eight basis and a four-node multi-class classifier (one background node and one per operator family), with an equal signal cross-section reweighting correcting the operator imbalance within each training category. Quantified as the Asimov significance relative to an $m_{JJ}$-only baseline under a $30\%$ Gaussian background uncertainty, the multivariate discriminants improve the expected significance by factors of $11$ to $281$ depending on operator and architecture. The reweighting proves decisive for the multi-class model, which reaches $79\%$ to $93\%$ of the specialised significance on the tensor family. All quoted significances assume a pure-EFT quadratic-only signal model with one operator active at a time. This is the first fully hadronic aQGC sensitivity study using a machine-learning discriminant in ATLAS.
\end{abstract}
